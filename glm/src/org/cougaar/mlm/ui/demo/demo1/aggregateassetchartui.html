Transportation Schedule

This illustrates the use of a java applet and a 3rd party charting package (JClass from the KL Group) to obtain information from a Plan Service Provider (PSP) and display that information. This applet is communicating with the COUGAAR mini-test configuration in order to obtain the aggregate assets owned by the 3ID cluster. The information is sent from the cluster to the applet as an XML stream. Currently, the only aggregate asset in the 3ID cluster in the mini-test configuration society is passengers; we expect that the data will be much more complete in the final society.
