Transportation Schedule

This illustrates the use of a java applet and a 3rd party charting package (JClass from the KL Group) to obtain information from a Plan Service Provider (PSP) and display that information. This applet is communicating with the COUGAAR mini-test configuration in order to obtain the schedules for asset usage in the MCCGlobalMode cluster. The information is sent from the cluster to the applet as an XML stream. Using the facilities of the JClass package, we can display different types of graphs (such as bar charts and scatter plots), and customize the graphs extensivley including customizing legends, axes, line, and data symbols and colors.
