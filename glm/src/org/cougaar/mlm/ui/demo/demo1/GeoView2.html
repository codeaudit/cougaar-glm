Geographic/Spatial Display Indexing

This applet illustrates the use of a Geographic Map or spatial display as an index into other displays. The following map is an imagemap which is partititoned into sub-regions. Selection of a sub-region on the main display will evoke a URL which can be displayed in separate browser window or in another frame on the current display.

This applet also illustrates the display and manipulation of simple sprites to "layer" additional information upon the spatial index. Control-MouseClick activates popup menu.
