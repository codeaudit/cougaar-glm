Supply

This illustrates the use of a java applet and the Swing JTable to display a stoplight chart for supply status. This applet is currently running off internally generated data, rather than communicating with the cluster society. The intent is that this applet could be used to communicate with a Plan Service Provider (PSP) in a cluster to obtain information about the supply status from a log plan. Position the mouse over the table and click the right mouse button to select different drill down capabilities.
