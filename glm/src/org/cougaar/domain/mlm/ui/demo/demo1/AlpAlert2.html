Active Alerts

This illustrates use of PSP (server) and HTML/Javascript (client) to provide continuous update to a browser console of Alerts. This particular model uses a tiny Java applet to listen to a PSP for alerts. As they occur it forces refresh of the left-side scrolling frame with the new Alert information appended. For this implementation, random Alerts are generated at PSP approximately every 10 secs and displayed via HTTP feed back to applet.
