Transportation Schedule

This illustrates the use of a java applet to present specific queries to obtain information from a Plan Service Provider (PSP) and display that information. This applet is communicating with the COUGAAR mini-test configuration in order to obtain information from a log plan. The information is sent from the cluster to the applet as an XML stream. The user can customize the queries by selecting items from the drop-down menus. Furthermore, the user can customize the display in several ways. For example, clicking in a column header of the table arranges the entries alphabetically or by date (depending on the information in the column). Also, holding and dragging the mouse in a column header, rearranges the columns.
