COUGAAR Policy Editor

This applet demonstrates the capability to view and edit COUGAAR policies. When a user selects a cluster, the applet connects to that cluster's UI Data PSP to retrieve the XML for all its policy objects. The user may select one of the policies by name from the "Policies" pull-down menu. Then the policy parameters and values appear in the table. To change a policy, the user simply clicks in the value field of a parameter and types in a new value. When all desired changes have been made to that policy, the user clicks Commit. At this point the edited policy is sent to the cluster for verification. If the new values are accepted, the update is complete and any plugins depending on that policy will use the new parameters from that point on.
