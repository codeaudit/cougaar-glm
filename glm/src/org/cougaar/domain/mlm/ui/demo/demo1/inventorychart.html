Inventory

This illustrates the use of a java applet and a 3rd party charting package (JClass from the KL Group) to display a chart showing due ins, due outs, and stockage level for a particular asset. This applet is currently running off internally generated data, rather than communicating with the cluster society. The intent is that this applet could be used to communicate with a Plan Service Provider (PSP) in a cluster to obtain information about the inventory from the log plan.
