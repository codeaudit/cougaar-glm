Overview of this Release

Build 9.0 is a major release in Cougaar development and the first one under the Ultra*log project.  Notable GLM additions for this release include:
  • In repackaging for new release, removed domain portion of package, and references to PlugIn
  • Addition of the readiness assessor
  • Improvement for UniversalAllocator to approve/disapprove any task matching a provided regular expression
  • Numerous Bug Fixes
Build 8.6 is a point release along COUGAAR 8.0 and will be the final release under the ALP project.  Future Cougaar releases will be under the Ultra*Log project. Notable GLM additions for this release include:
  • Enhancements to the generic UniversalExpanderPlugin which expands tasks according to sub-tasks defined in a database table. Enhancements include better ability to defined task parameters, ability to specify more than 2 levels of expansion, and the ability to expand levels fully or expand them into a single flat workflow. For details please see UniversalExpanderPlugin.html
Build 8.2 is a new major release of COUGAAR. The GLM (Generic Logistics Model) module has LDM and Plugins with a logistics domain focus. This is the first release where the release notes are documented seperately from the rest of Cougaar. Essential differences between 7.2 and 8.2 include: 
  • Streamlined Inventory Manager for better performance
  • Reorganized Inventory Manager operations for better design and more accurate solutions
  • Multiple bug fixes.
